# Content Spec: 카테고리 & 체크리스트 항목

## Overview

미국 이주 준비에 필요한 체크리스트를 8개 카테고리, 66개 항목으로 구성한다.

## 기존 대비 변경사항

### 제거된 카테고리
- `LOGISTICS` ("이사 & 생활") — 항목들이 MEDICAL, TRANSPORT, KOREA_DEPARTURE로 분산

### 신규 카테고리 (4개)
- `DOCUMENTS` — 법적/행정 서류 (아포스티유, 증명서 등)
- `MEDICAL` — 의료/건강 (보험, 검진, 처방약 등)
- `TRANSPORT` — 교통/자동차 (면허, 차량, 보험 등)
- `KOREA_DEPARTURE` — 한국 출국 준비 (행정 정리, 이사, 해지 등)

### 네비게이션 변경
- 기존: Bottom Navigation (Dashboard + 5 카테고리 탭)
- 변경: Dashboard 허브 방식 (카테고리 카드 클릭 → 체크리스트 이동)
- 이유: 8개 카테고리는 Bottom Navigation에 부적합 (Material 3 권장 3-5개)

### DB 변경
- `AppDatabase` version 1 → 2
- `fallbackToDestructiveMigration()` 적용 (pre-release 단계)
